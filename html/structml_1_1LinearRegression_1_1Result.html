double ml::LinearRegression::Result::r2 ( ) const

R2 coefficient.

1 - fraction of variance unexplained relative to a "base model", estimated as population variance. Equal to 1 - rss / tss.

◆ adjusted_r2()

double ml::LinearRegression::Result::adjusted_r2 ( ) const
inline

Adjusted R2 coefficient.

1 - fraction of variance unexplained relative to a "base model", estimated as sample variance. Equal to 1 - rss * base_dof / tss / dof.

double ml::LinearRegression::Result::tss

Total sum of squares. For fitting with intercept: \( \mathrm{TSS} = \sum_{i=1}^N (y_i - N^{-1} \sum_{j=1}^N y_j)^2 \). For fitting without intercept: \( \mathrm{TSS} = \sum_{i=1}^N y_i^2 \).
